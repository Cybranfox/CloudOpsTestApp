## ğŸš€ Getting Started

### Prerequisites
- Python 3.11+
- Node.js 18+ (for mobile build)
- Android Studio (for APK development)

### Quick Start
```bash
# Clone the repository
git clone https://github.com/Cybranfox/CloudOpsTestApp.git
cd CloudOpsTestApp

# Install dependencies
pip install flask flask-cors

# Run the application

```
